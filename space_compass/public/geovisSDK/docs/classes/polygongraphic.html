perPositionHeight

  • get perPositionHeight(): boolean
    • Defined in LayerManager/GraphicLayer/GraphicItem/PolygonGraphic.ts:174
    只读

    只有初始化是可以设置该属性（clampToGround为true时无效）

    描述

    指定是否使用每个点的高度，一般需要把 extrudedHeight 参数设置为0

    参考示例
 let polygon = new GV.PolygonGraphic({
    positions:[new GV.GeoPoint(0,10,5000), new GV.GeoPoint(0,20,200000),new GV.GeoPoint(10,20,300000),new GV.GeoPoint(10,0,200000)], //设置坐标
    extrudedHeight : 0,
    perPositionHeight : true
 });
 viewer.graphicLayer.add(polygon); //添加到场景

    Returns boolean
