6       系统服务管理

在GEOVIS iCenter系统服务管理中可以管理GEOVIS iCenter所有的系统服务。包含：

l  Register 注册服务

l  Config 配置服务

l  Auth 认证服务

l  Permission 权限服务

l  Geoserverplus Geoserverplus

l  Image 影像管理服务

l  Webmanager Web管理器

l  Vector 矢量管理服务

l  Dm 数据管理服务

l  Place 地名服务

l  Gateway 网关服务

 

l  如何登录到系统服务管理？

 在浏览器中输入：http://<服务器IP>:8100/ , 可以打开GEOVIS iCenter系统服务管理的登录界面。

 输入系统服务管理器的默认管理员账户进行登录：用户名：admin，初始密码：ssmanager。

l  如何启动系统服务？

进入系统服务管理后，点击页面上方的“全部启动”按钮，系统会按照默认启动顺序依次启动所有系统服务。所有服务的服务状态变为绿色则启动成功。

也可以手动启动服务，直接点击服务右侧的启动服务按钮即可。但需注意，手动启动服务时，必须按照默认启动顺序依次启动服务，否则可能无法启动成功。

l  如何更新系统服务？

在服务右侧，点击更多操作，在下拉列表中选择更新服务，会弹出的对话框中选择新的服务包，点击确定。
